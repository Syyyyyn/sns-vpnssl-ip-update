# sns-vpnssl-ip-update

Python tool for Stormshield VPN SSL IP update

## Getting started

### Prerequisites

**Running environnment**

This application requires Python3 and Paramiko in order to be executed.

- Python3

```
$ apt install python3 python3-pip
```

- Paramiko module, for automated SSH connections

```
$ pip3 install paramiko
```

**Dependencies**

This application uses `sshpass` to download & upload configuration to the appliance.

```
$ apt install sshpass
```

### Installation

```
$ git clone https://github.com/Syyyyyn/sns-vpnssl-ip-update.git
```
